# Chirper Customizado (Laravel)

Este projeto é uma versão estendida do **Chirper**, aplicação oficial de exemplo do Laravel, com diversas melhorias funcionais, estruturais e de experiência do usuário.

---

# Principais Melhorias Implementadas

## Página de Perfil

A aplicação agora conta com uma página de perfil dedicada para cada usuário.

### Funcionalidades:

- Exibição de informações do usuário
- Foto de perfil (avatar)
- Listagem dos chirps do usuário
- Interface personalizada por usuário

---

## Botão Home

O botão **Home** permite ao usuário visualizar o feed principal.

### Funcionalidades:

- Exibição de todos os chirps (ou feed geral) de usuários que você segue
- Atualizações em tempo real após interações

---

## Botão Explore

O botão **Explore** foi implementado para descoberta de conteúdo.

### Funcionalidades:

- Visualização de chirps de outros usuários (além dos que você segue)

---

## Sistema de Follow

Foi implementado um sistema de seguidores entre usuários.

### Funcionalidades:

- Seguir outros usuários
- Deixar de seguir (unfollow)
- Relacionamento entre usuários (followers/following)

---

## Sistema de Like

Os usuários agora podem curtir chirps.

### Funcionalidades:

- Curtir e descurtir publicações
- Contagem de likes por chirp
- Feedback visual da interação

---

## Busca de Usuários

Foi implementado um sistema de busca para encontrar outros usuários na plataforma.

### Funcionalidades:

- Pesquisa por nome de usuário
- Exibição de resultados em tempo real ou via requisição
- Acesso rápido ao perfil de outros usuários

---

## Sobre o Projeto Original

O **Chirper** é uma aplicação simples estilo microblog (semelhante ao Twitter), utilizada para demonstrar conceitos do Laravel como:

- Autenticação
- CRUD básico
- Autorização
- Blade Templates
- Eloquent ORM
